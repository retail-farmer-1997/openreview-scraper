## Planning Policy
- Small tasks: lightweight plan in the PR description or issue.
- Large tasks: execution plan document under `docs/exec-plans/active/`.
- Completed work: move the final plan to `docs/exec-plans/completed/`.

## Required Plan Sections
- Objective
- Scope
- Risks
- Validation
- Status Log

## Drift Control
